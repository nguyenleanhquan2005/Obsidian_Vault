## Cách xử lý

### Overfitting (Variance cao)
- Thêm data
- [[Regularization Algorithms]] (L1, L2)
- Giảm độ phức tạp model
- Dropout (Deep Learning)
- Early stopping
- [[Cross Validation]]

### Underfitting (Bias cao)
- Tăng độ phức tạp model
- Thêm features ([[Feature Engineering Overview]])
- Giảm regularization
- Train lâu hơn
