## 🏫 EXAM 1 – STUDENT CLUB MANAGEMENT SYSTEM

**Scenario:**
FPT University wants to build an online **Student Club Management System (CMS)**. The system must allow students to register for clubs, allow club administrators to track activities and attendance, and allow the university to monitor the overall picture. The project lasts **4 months**, with a budget of **$12,000**, and must have at least **200 active student users within the first month** after launch. The system must achieve a **system uptime ≥ 99.5%** and a response time of **< 2 seconds** per request.

---

**Request 1 (20%):** Identify **at least 5 stakeholders** for this project. For each stakeholder, provide:
- Name / Role
- A detailed description of their specific responsibilities in this project
- Classification: Internal or External
- Their position on the Power/Interest Grid and the corresponding management strategy

---

**Request 2 (20%):** Create a **RACI Matrix** with at least 10 tasks/activities. The roles to include in the matrix are: Project Manager (PM), Business Analyst (BA), Developer (Dev), Tester (QA), and Product Owner (PO).

> [!tip]
> Each task must have exactly **1 "A"**. Every task must have at least **1 "R"**. Use all four symbols: R / A / C / I.

---

**Request 3 (30%):** Write **3 SMART Goals** for this project. Analyze each goal fully using the S–M–A–R–T criteria (one criterion at a time).

---

**Request 4 (30%):** Identify **at least 3 project risks**. For each risk, provide:
- Risk Name & Description
- Probability (Low / Medium / High) & Impact (Low / Medium / High)
- Mitigation Plan (actions to reduce the likelihood or impact before the risk occurs)
- Contingency Plan (actions to take if the risk has already occurred)

## 📌 ANSWER KEY – EXAM 5 (EVM Calculations)

> [!note]
> This is a reference answer. Try solving it yourself first before checking!

**Given:** BAC=$120,000 | BDAC=12 months | Evaluation at Month 6 | AC=$55,000 | % Complete=40%

| Metric | Formula | Result | Conclusion |
|--------|---------|--------|------------|
| PV   | 120,000 × (6/12) = | **$60,000** | Planned to spend $60k by Month 6 |
| EV   | 120,000 × 40% = | **$48,000** | Value of work actually accomplished |
| CV   | EV − AC = 48,000 − 55,000 = | **−$7,000** | ❌ OVER BUDGET |
| SV   | EV − PV = 48,000 − 60,000 = | **−$12,000** | ❌ BEHIND SCHEDULE |
| CPI  | EV / AC = 48,000 / 55,000 = | **0.87** | Every $1 spent yields only $0.87 in value |
| SPI  | EV / PV = 48,000 / 60,000 = | **0.80** | Project is running at 80% of planned speed |
| EAC  | BAC / CPI = 120,000 / 0.87 = | **~$137,931** | Final cost will exceed budget by ~$17,931 |
| ETC  | EAC − AC = 137,931 − 55,000 = | **~$82,931** | Still needs ~$82,931 to finish |
| VAC  | BAC − EAC = 120,000 − 137,931 = | **−$17,931** | Expected to exceed budget by $17,931 |
| EDAC | BDAC / SPI = 12 / 0.80 = | **15 months** | Project will finish 3 months late |

**Overall Assessment:** The project is **OVER BUDGET** (CPI=0.87 < 1) and **BEHIND SCHEDULE** (SPI=0.80 < 1). Immediate corrective action is required!
